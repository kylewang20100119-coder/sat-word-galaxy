Lexiverse 不是一张从 A 排到 Z 的单词表。

点开一个词，你会看到它的同义词、反义词、词源、前缀和同源词；继续学习，系统会记住你的熟悉度、学习遍数、错题和阅读进度。第一遍认识单词，第二遍建立词义网络，第三遍把它放回 Digital SAT 风格的文章中。最终目标不是“这个词我见过”，而是阅读时能迅速理解它在句子里的作用，并抓住整篇文章的逻辑。

> 不知道从哪里开始？打开网页后，直接点击 **“带我开始”**。系统会根据当前 Group、弱词、到期复习和今日进度，给出一个最小、明确的下一步。

## 第一次使用，只做这三步

### 1. 选择正在背的 Group

从左侧导引栏进入 **Group 背词**，选择你的 SAT Group 和当前遍数。如果你刚开始使用，就选 **第一遍**。

### 2. 每看完一个词，给出真实评级

学习卡顶部有一组 `1–5` 熟悉度按钮。不要为了数字好看而高估自己：选完后，系统会保存评级、记录这一遍学习，并自动进入下一个词。

| 等级 | 什么时候选它 |
| --- | --- |
| L1 · 陌生 | 几乎不认识，看到后没有印象 |
| L2 · 模糊 | 见过，但想不起准确意思 |
| L3 · 想起 | 能想出主要意思，反应还不够快 |
| L4 · 掌握 | 能在句子中判断意思和语气 |
| L5 · 秒反应 | 看到后几乎不需要翻译就能理解 |

也可以直接按键盘 `1–5` 评级，用 `←`、`→` 浏览前后单词。

### 3. 先完成 10 个，不要求一口气完成 100 个

每天的完整目标是 100 个不同单词，但第一段只需要完成 10 个。系统会记录今日连击、XP、Group 进度和学习断点。进入节奏以后，再决定继续到 20、50 或 100。

## 单词星系怎么用

### 搜索一个单词

在顶部固定搜索栏输入英文单词。结果按字母顺序排列，也可以搜索中文意思和英文释义。选择结果后，这个词会成为星系中心。

### 看懂不同关系

可以分别打开或关闭：

- **同义词：** 意思相近，距离越近通常代表语义越接近。
- **反义词：** 用相反方向帮助确定一个词的边界。
- **词源：** 看不同单词如何从相同历史来源发展出来。
- **前缀：** 比较共享构词部分的单词。

星系只绘制中心词附近最有用的一圈关系，不会一次把全部词库塞进画布，因此在词量很大时仍能保持可读性。

### 从学习卡联动星系

在 Group 背词区切换下一个词时，上方星系会同步切换中心词。你也可以点击学习卡里的关联词，把它展开到上方星系；当前背词卡不会因此跳走，方便对比后继续原来的进度。

## 一个单词能看到什么

每张单词资料卡会尽可能提供：

- 单词的全部常见词性，以及每个词性分别对应的意思
- 英文解释与中文意思
- 发音
- 例句和固定搭配
- 同义词、反义词，以及它们“相同到什么程度”
- 词源与构词拆解
- 和词源有关的记忆方法
- 同源词及各自的意思
- 当前熟悉度与累计学习遍数

词源优先使用可靠的拉丁语来源。单词并非来自拉丁语时，会如实标注希腊语、法语、古英语或其他来源，不会为了记忆方便硬套词根。

## 三遍背词是怎样工作的

三遍不是把同一张卡机械看三次，而是三种不同任务。系统会分别保存每一遍的 Group、词位、完成情况和日期。

### 第一遍：完整认识

适合第一次接触这个 Group。

重点看：

1. 词性和核心意思
2. 例句与常见搭配
3. 词源和记忆钩子
4. 自己真实的熟悉度

目标是建立第一印象，不要求立刻掌握所有细节。

### 第二遍：建立词义网络

第二遍不做阅读。重点是把一个词和它周围的词连接起来：

1. 先回忆意思，再展开答案。
2. 看当前词在例句中的使用方式。
3. 比较同义词之间的强弱、语气和适用场景。
4. 用反义词确定语义边界。
5. 用同源词理解词根如何产生一整个词族。

低熟悉度或回忆失败的词会进入记忆救援队列，隔开几个词后闭卷重现。答错不会扣 XP，也不会中断今日连击。

### 第三遍：回到文章，训练结构

第三遍才进入阅读。

每个 Group 的全部单词会被分配进 **3 篇精修文章**，一般按照 `3 / 3 / 4` 的方式覆盖，并优先把语义相近或适合共同出现的词放进同一篇文章。每篇只做一道题，不追求题海，重点训练：

- 作者正在做什么：提出观点、描述研究、修正旧解释，还是比较两种立场？
- 哪个词或句子构成 pivot，也就是文章逻辑的转折点？
- 哪一句是真正的结论，哪些只是背景或证据？
- 即使有个别生词，能否先抓住结构并排除错误选项？

目标词不会在正文中高亮。答题后才会展开文章骨架、转折作用、选项分析和一句话复盘。完成记录会进入 **阅读记录**，可以随时回看。

> 推荐顺序：一个 Group 的第二遍完成后，再开始它的第三遍。二刷负责建立词义关系，三刷负责让这些关系在文章里自动反应。

## 智能复习怎么用

进入 **智能复习** 后，可以直接选择 L1–L5 中任意熟悉度，连续复习对应单词。系统会优先安排已经到达复习时间的词，并在原句和使用场景中重新激活记忆。

一次有效复习不是重新扫一遍释义，而是：

1. 先读场景，判断这个词在句中的意思和作用。
2. 再查看例句、搭配和词源记忆钩子。
3. 根据真实感觉判断：只是认得意思，还是已经能还原使用场景。

记得的词会逐步延长下次出现的间隔；忘记的词会更快回来。你也可以使用 **复习今天背过且熟悉度未达到 L5 的单词**，在当天结束前补一轮短复习。

薄弱词高难度阅读会优先选取当前 L1–L2 单词，让复习不止停留在孤立释义上。

## 易混词对比怎么用

易混辨析不是让你再读两张单词卡，而是先找边界，再做选择。

1. 选择“只看当前 Group”或浏览完整易混词库。
2. 阅读两个词最关键的差别。
3. 在真实语境中完成二选一。
4. 看完解析后，诚实选择“还会混”或“已经分清”。

点击任意词名，可以在上方星系中展开它，同时保留当前辨析题。系统会记录今日练习数量、已经分清的词对和仍需复习的薄弱组合。

## DSAT 训练怎么用

**DSAT 训练** 是独立的离线单题题库，适合在日常背词之外检查阅读能力。

可以按照以下条件筛选：

- SAT Group 范围
- 熟悉度
- 文章类型与学科
- 难度
- 是否显示目标词意思

当前题库包含 250 道离线训练题，覆盖词汇语境、主旨、推断、证据、篇章目的和跨文本联系等题型。系统会优先避开已经做过的题目，并保存每次作答、正确率和错题。

Group 三刷阅读和 DSAT 训练的作用不同：

- **Group 三刷阅读：** 用指定 Group 的全部词训练文章结构和语境反应。
- **DSAT 训练：** 用不同话题与题型检查综合做题能力。

## 学习仪表盘与游戏系统

学习仪表盘会把进度拆成真正有用的几层，而不只是“背过 / 没背过”：

- 每个 Group 的一刷、二刷、三刷完成度
- L1–L5 熟悉度分布和平均熟悉度
- 每个词累计学习遍数
- 今日学习词数与连续学习天数
- 今日阅读、易混辨析和语境复习数量
- 累计 XP、可用 XP、段位、星图碎片和勋章

### 每日节奏

- 20 词：完成热身
- 50 词：进入巡航
- 100 词：完成当天目标
- 每完成 10 词：获得一次跃迁奖励

### 用 XP 兑换休息

可用 XP 可以兑换 5、15 或 30 分钟休息。兑换后会出现全屏半透明倒计时；时间结束，系统会提醒你回到学习。兑换只减少可用 XP，不会降低累计 XP、段位或已经解锁的勋章。

XP 是节奏工具，不是考试分数。真正重要的数据仍然是熟悉度、三遍进度、阅读结构识别和错题记录。

## 一个可直接照做的每日流程

如果功能很多，不需要每个模块都用一遍。按照当前阶段选一条主线即可。

### 正在一刷

```text
10 词热身 → 继续当前 Group → 当天达到 20 / 50 / 100 词 → 复习今日未到 L5 的词
```

### 正在二刷

```text
先回忆意思 → 看例句 → 比较同反义词 → 扩展同源词 → 完成记忆救援
```

### 正在三刷

```text
完成一篇 Group 阅读 → 先判断文章骨架 → 再答题 → 复盘 pivot 和结论 → 回看错题
```

如果当天状态很差，只完成系统给出的 **10 词冲刺**。保持启动，比在开始前要求自己完美完成 100 个更重要。

## 学习记录与备份

熟悉度、三遍进度、XP、连击、错题和阅读记录都保存在当前浏览器中。

以下操作不会让记录消失：

- 刷新网页
- 关闭网页后重新打开
- 在本地按 `Control + C` 停止服务
- 在同一网址更新网页代码

以下情况可能无法自动读取原记录：

- 清除浏览器的网站数据
- 使用无痕模式
- 更换浏览器
- 更换电脑、手机或 iPad
- 从一个网址换到另一个网址

### 把记录带到另一台设备

1. 进入 **学习仪表盘**。
2. 找到 **进度保险箱**。
3. 下载完整备份。
4. 把备份文件发送到另一台设备。
5. 在新设备的进度保险箱中选择恢复。

备份包含熟悉度、三遍进度、XP、连击、错题、阅读和易混词记录。文件只在你的浏览器中生成和读取，不会上传到服务器。

当前版本没有账号和云数据库，因此电脑与 iPad 不会自动同步。建议在完成一个 Group、进行大量评级或更换设备前下载一次备份。

## 常见问题

### 我应该从星系、Group 背词还是复习开始？

日常主线从 **Group 背词** 开始。星系用于理解关系，智能复习用于处理到期词和弱词，DSAT 训练用于检查综合阅读能力。

### 为什么上方星系变了，下方背词卡没有变？

这是有意设计的。学习卡切换会联动星系；但在星系里探索关联词时，不会打断下方正在背的词。

### 为什么第二遍没有阅读？

第二遍的任务是用意思、例句、同反义词和同源词建立词义网络。第三遍才把单词放入文章，训练阅读中的快速反应。

### 为什么第三遍的目标词没有高亮？

三刷模拟真实阅读。你需要在正常版面中认出单词，并优先抓住主旨、转折和结论。答题后才会看到结构反馈。

### 评级后还能修改熟悉度吗？

可以。再次打开这个词并重新评级即可，星系资料卡和 Group 学习使用同一份熟悉度数据。

### 刷新网页后连击和进度会丢吗？

不会。它们会保存在当前浏览器。清除网站数据或换设备前，请先下载完整备份。

### 需要 OpenAI API Key 吗？

不需要。单词星系、三遍背词、智能复习、Group 阅读和离线题库都可以直接使用。

### 题目是 SAT 官方真题吗？

不是。所有练习内容都是本项目的原创学习材料，用于模拟 Digital SAT 的文本长度、推理方式和选项结构。
