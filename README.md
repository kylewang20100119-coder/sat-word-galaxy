# SAT + GRE Word Galaxy

一个无需安装依赖、可以在本地运行的 SAT + GRE 单词星系与新版 Digital SAT 备考网页。模拟题默认从原创离线题库读取，不需要 API Key。

## 在 VS Code 中运行

在这个文件夹中运行：

```bash
npm run dev
```

然后打开 <http://localhost:8000>。单词星系、离线模拟题和复习系统都可以直接使用，不消耗 API 额度。

## 可选的 OpenAI 接口

项目保留了实验性的 OpenAI 文章接口，但默认网页不会调用它。如需自行开发实时生成功能，可在 macOS / Linux 的终端运行：

```bash
export OPENAI_API_KEY="你的 OpenAI API Key"
npm run dev:api
```

然后打开 <http://localhost:8000>。API Key 只由本机的 `server.py` 读取，不会写进网页文件，也不会发送给浏览器。

默认使用 `gpt-5.4-mini`。如需切换模型，可以在启动前设置：

```bash
export OPENAI_MODEL="gpt-5.4-mini"
```

如果端口 8000 已被占用：

```bash
PORT=8001 npm run dev
```

> 日常使用只需运行 `npm run dev`，无需配置 API Key。

## 已实现

- 星系式单词空间网络，支持拖动、缩放、文字避让与平滑转场
- 精简、标准、丰富三档密度；默认仅标注核心邻居，外围关系用星点表示
- 同义词、反义词、词源、前缀四类关系独立开关
- 全部、SAT、GRE、SAT + GRE 重合词四类词库筛选
- 保留 SAT Excel Group 1–179，并新增 GRE Core / GRE Advanced 分类
- 关系越强，节点排列越近；不相关的词不连线
- 搜索英文、中文意思和英文释义
- 自定义单词范围，仅显示指定词汇和范围内关系
- 英文解释、中文意思、例句、固定搭配、词源、词源记忆法、词性和可对照学习的同源词族
- 唯一一组 1–5 级熟悉度评级，支持数字键、自动前进与星系详情双向同步，并自动保存在浏览器本地
- 每个 SAT 单词记录累计学习遍数；顺序背词、复习卡片和阅读训练都会持续累加
- 45 组 SAT / GRE 高频易混词对比背诵舱，包含核心区别、双例句、记忆钩子、薄弱词队列与分清记录
- 二刷、三刷和复习旧词同样计入每日 100 词目标并获得 XP；每日按不同单词去重，避免重复刷同一个词虚增进度
- 20/50/100 词阶段奖励，以及复习卡、易混词、阅读三类跨模式每日任务
- 每 10 词完成一次跃迁并获得星图碎片；每 5 枚碎片点亮一个可收藏星座，共 12 个星座
- 四阶段征服路线分别统计完成 1、2、3、4 遍的单词数量，并配有 18 枚自动解锁勋章和终极三刷毕业目标
- Group 阅读支持“二刷重点高亮”和“三刷原文直读”；三刷不再高亮目标词，专门训练在原文中直接反应词义
- 每次 Group 阅读会保存题目快照、目标词、作答结果和日期，并统计当天篇数、当前 Group 篇数、总量与正确率
- 按 SAT Group、熟悉度、文章类型和难度匹配原创 Digital SAT 短文本单题
- 当前离线题库共 250 题：40 道手工多题型仿真题、90 道词汇语境题和 120 道阅读理解题；新增部分以 840 个不重复 SAT 词为优先，覆盖自然科学、社会科学、人文研究与文学叙事
- 题目采用 25–150 词左右的短文本与一题四选一结构，覆盖推断、主旨、证据、语境词义、篇章目的和跨文本联系
- 目标词会在文章中高亮，并附带语境释义和答题解析
- 按熟悉度与到期时间生成复习队列；“记得”会逐级延长复习间隔，“忘记了”会降低熟悉度并尽快重现
- 离线题库会避开已经做过的题目，不调用 API；手工仿真题位于 `offline-questions.js`，词汇覆盖扩充逻辑位于 `generated-questions.js`
- 提交答案后永久从正常抽题池排除，并保存做题记录、正确率和错题；可随时重新查看原题、作答结果与解析
- 深色/浅色主题及移动端布局

## 扩展词库

精选扩展词位于 `words.js`，Excel 导入数据位于 `wordbank.js`，GRE 基础数据位于 `gre.js`，本轮新增的 GRE 关系词、同源词族和词源补全位于 `lexical-expansion.js`：

- `window.WORDS` 中添加单词详情
- `window.RELATIONS` 中添加 `[起点, 终点, 关系类型, 相似度]`
- 关系类型可用：`synonym`、`antonym`、`etymology`、`prefix`
- 相似度使用 `0` 到 `1`，越大代表关系越接近

当前包含 1,798 个 SAT 词和扩展词、115 个带完整释义和学习属性的 GRE 阅读高频/高难词。系统识别出 552 个 SAT + GRE 重合词，因此总词数为 1,912，GRE 分类共覆盖 666 个词。搜索覆盖完整词库，画布仅按需绘制当前单词附近的有限节点，避免一次渲染全部词汇。

GRE 选词优先补充能与现有 SAT 词形成明确同义、反义或同源关系的高难阅读词；中文释义、英文释义、例句、搭配和词根记忆内容为本项目重新整理。新增完整 GRE 词均至少拥有一条可渲染关系，不会成为星系中的孤立空壳。

大部分词条的英文释义、中文释义、例句、搭配、发音和词源来自本机词典数据。新增词源优先标注拉丁语；没有拉丁来源时会明确标注希腊语、法语、古英语、苏格兰英语或人名来源。仍无法可靠确认的条目只显示透明构词线索并注明需要继续核对，不会硬套拉丁词根。
